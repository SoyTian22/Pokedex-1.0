# Daniel Cristian GovTech Lab - Motion v9

Proyecto preparado por Cristian Montiel para presentar a Daniel una demo GovTech con diseño oscuro premium, mapa departamental, 50 soluciones/capas, diagnostico guiado, aula interactiva, graficas y transiciones avanzadas.

## Link publico activo

https://ea8f1c6ce40b531dd6.v2.appdeploy.ai/

## Mejoras de motion design aplicadas

- CSS View Transitions API para cambios de pantalla.
- transition-property, transition-duration, transition-timing-function y transition-delay en tarjetas, navegacion y departamentos.
- Easing: ease, linear, ease-in-out y cubic-bezier(.16,1,.3,1).
- Transformaciones con translate, scale, blur, clip-path y saturate.
- Efecto smoke/glow en fondo negro.
- Tarjetas con hover neon y efecto de profundidad.
- Navegacion tipo capsula/animated navbar para mobile.
- Mapa departamental con iluminacion por partido politico.
